PEDALBOARD PLUGIN PARAMETER REFERENCE
=====================================

Generated from the VST3 files currently loaded by process.py:

    noise      = pb.load_plugin("rnnoise.vst3")
    eq         = pb.load_plugin("FreeEQ8.vst3")
    compressor = pb.load_plugin("ZL Compressor.vst3")
    gate       = pb.load_plugin("BPGate.vst3")

The values marked "initial" are the values reported immediately after loading the
plugin. Pedalboard does not expose a separate default_value property for these
plugins, so the initial value is the closest useful equivalent.


HOW TO SET PARAMETERS
---------------------

Use the parameter ID, not the display name:

    set_parameter(gate, "threshold_db", -24)
    set_parameter(compressor, "threshold_db_threshold_db", -18)
    set_parameter(eq, "band_1_freq", 100)
    set_parameter(noise, "vad_threshold", 0.6)

Direct Pedalboard assignment also works when the value is already exact:

    gate.attack_ms = 10.0
    compressor.compressor_style_compressor_style = "Optical"
    eq.band_1_type = "HighPass"
    noise.bypass = False

To inspect a plugin at runtime:

    print(plugin.parameters.keys())

    for param_id, info in plugin.parameters.items():
        print(
            param_id,
            info.name,
            info.type,
            info.min_value,
            info.max_value,
            info.valid_values,
            getattr(plugin, param_id),
        )

Notes:

* String choices are case-sensitive when assigned directly.
* set_parameter() in process.py also handles normalized strings and maps a
  numeric input to the nearest numeric-looking string choice.
* Boolean values should be True or False.
* Float ranges are inclusive.
* Some plugins expose hundreds of quantized values for a float. Supplying a
  normal number inside the documented range is sufficient.


1. RNNOISE SUPPRESSION FOR VOICE
================================

File: rnnoise.vst3
Pedalboard name: RNNoise suppression for voice
Parameter count: 4

vad_threshold
    Display name: VAD Threshold
    Type: float
    Range: 0.0 to 1.0
    Initial: 0.6
    Meaning: Voice activity detection threshold.

vad_grace_period_10ms_per_unit
    Display name: VAD Grace Period (10ms per unit)
    Type: float, whole-number steps
    Range: 0 to 500
    Initial: 20
    Meaning: Grace period in units of 10 ms. A value of 20 is 200 ms.

retroactive_vad_grace_period_10ms_per_unit
    Display name: Retroactive VAD Grace Period (10ms per unit)
    Type: float, whole-number steps
    Range: 0 to 10
    Initial: 0
    Meaning: Retroactive grace period in units of 10 ms.
        

bypass
    Display name: Bypass
    Type: bool
    Choices: False, True
    Initial: False


Example: 

    set_parameter(noise, "vad_threshold", 0.6)
    set_parameter(noise, "vad_grace_period_10ms_per_unit", 20)
    set_parameter(
            noise,
            "retroactive_vad_grace_period_10ms_per_unit",
            5,
        )
    Example: set_parameter(noise, "bypass", False)

    Initial: 1000.0

    set_parameter(compressor, "filter_status0_filter_status", "On")
    set_parameter(compressor, "filter_type0_filter_type", "High Pass")
    set_parameter(compressor, "order0_order", "24 dB/oct")

    set_parameter(compressor, "gain0_gain", 0)
    set_parameter(compressor, "q0_q", 0.707)

    Display name: Bypass
    Type: bool
    Initial: False

        compressor,
